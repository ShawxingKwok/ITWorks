KLog

abstract class KLog(defaultTag: String, isDebug: Boolean)

A log util extended from android logcat.

Usage example:

// Set your log class.
object MLog : AndroidKLog("Shawxing", BuildConfig.DEBUG)

// Log somewhere.
MLog("Jack")
MLog.i(20, 1.8)

// Logs:
D    Jack
         at ...(xx.kt:10)
I    20, 1.8
         at ...(xx.kt:11)
Content copied to clipboard

Note: for released android libraries, on level 'VERBOSE' and 'DEBUG', its logs are not visible; and on level 'INFO' and 'DEBUG', traces are not visible.

If you are developing an application, remember to set '-assumenosideeffects class pers.shawxingkwok.android.KLog{ *; }' in 'proguard-rules.pro' in the app module to remove logs in the released apk.
