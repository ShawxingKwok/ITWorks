            迁移 (Migration) #

            KDataStore 内置 protected context 供你获取其他存储仓库，如 SharedPreferences, MMKV, DataStore 等。

            这份中文版本主要为初期的宣传考虑，其他作品并不会写中文版本和 Java 扩展。
            关于其他地方的存储方案分析，绝大多数对比都严重遗漏和偏颇，大多数测试都有严重错误。 官网相对准确，但经常掩盖缺点，建议配合 ChatGpt 查阅分析。
